Examples on how to R ang Geogebra can interact

For these examples we have used R package OpenCPU, which allows R to behave as a web server. The GeogebraR package (with source code and examples) is available at GitHub.

The examples have been tested on Chrome and Mozilla. For some reason, the examples do not seem to run on Safari.

  • Simple example to sample random points.
  • Quality Control Charts.
  • Some maps and city locations from Spain.
  • Fitting non-linear curves to a set of points.
